## Hoppscotch scripts
Generated during `npm run setup-sign-in-hoppscotch` (stored in `output/hoppscotch-sync/`):
- Uses **Hoppscotch `pw` namespace** (not Postman `pm`)
- Open collection: **TEST → sign-in** (not the old root `Sign In` folder)
- Select environment: **Sign In - Test** (required for `<<BASE_URL>>` variables)

## One-click run in Hoppscotch UI
1. Select team environment **Sign In - Test**
2. Open **TEST → sign-in** collection
3. Click **Run collection** — all requests execute with pre/post scripts and pass/fail results
